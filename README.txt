  Este Trabalho Prático de Matemática Discreta foi 
desenvolvido por Luiz Henrique Ferreira Neves e 
Danilo Araújo Domingues Silva. Todos os desafios 
enfrentados durante este trabalho foram solucio-
nados consultando o material disponível pelo pro-
fessor de Matemática Discreta do Instituto Fede-
ral de Minas Gerais do campus Formiga, o IFMG,
além das dúvidas que foram retiradas consultando
a internet, com vídeos e textos tutoriais acerca
da utilização de variadas funções e ferramentas 
da linguagem C de programação. 
  O editor de código utilizado para a escrita de 
todo o código desenvolvido para o trablaho
prático foi o Visual Studio Code, porém, para 
a execução de um trabalho mais eficiente e de for-
ma a nos ajudarmos, usamos uma extensão disponí-
vel para o Visual Studio Code chamada Live Share,
que permite que uma ou mais pessoas possam editar
um mesmo código ao mesmo tempo, à distânica, eco-
nomizando tempo e facilitando a demonstração do 
funcionamento do código desenvolvido. O sistema
operacional que utlizamos é Microsoft Windows e
o compilador utilizado foi o GCC.
